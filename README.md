## :bulb: Getting Started

#### Clone respository
```
$ git clone https://github.com/BoboLin945/trash-talk-generator.git
```
#### Install by npm
```
$ npm install
```
#### Execute
```
$ npm run dev
```
#### Browse the website on localhost
```
http://localhost:3000/
```
